User Guide

About

Upstream is a web assistant powered by OpenAI API and Chrome API that helps users process information faster by using their browser as context. The focus is on increasing productivity of day to day tasks that we all do on the web: things like writing an email, researching items to purchase, writing bulleted summaries of long articles, generating tables from text, and many more.

Usage

Upstream is a Chrome extension that can be used with any tab. To use it, simply click on the Upstream icon in the top right corner of your browser to open the interface. Then, ask questions of what you want to do with your current tab as context, here are some examples:

  • Write a one liner about this topic.
  • Extract the event timeline from this articles into CSV format.
  • Generate a tweet describing this article.
  • Explain the problem described.

How does it work?

Upstream uses OpenAI API to generate text, and Chrome API to access the current tab.

Privacy

Upstream does not collect any data from users. Only the text that you submit is sent through our servers and OpenAI's API. Upstream uses OpenAI API to generate text, and Chrome API to access the current tab.

Upstream is not liable for content generated by OpenAI API.
