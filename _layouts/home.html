Use GPT on Chrome

User Guide

About

Upstream is an Chrome extension that helps users save time on daily tasks. Things like writing an email, researching items to purchase, summarizing long reads, creating excel tables from text, and much more.

Upstream uses OpenAI API to generate text and Chrome API to access your tab's text.

See demo.

Usage

To use it simply open the extension and give Upstream instructions using your current tab, here are some examples: 

  • Write a one line summary. 
  • Extract the event timeline into CSV format. 
  • Generate a tweet describing this article.
  • Explain the problem described.

Upstream reads the tab you are on as context. This means you don't need to be specific, it already knows what you refer to.

Privacy

Only the text that you submit is sent through our servers and OpenAI's API. You can see at all times what is being sent by clicking the `context` tab.
